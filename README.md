# 🌌 NexusTale: AI 驱动的无限文字冒险引擎

**欢迎来到 NexusTale。在这里，没有剧本，只有未知的地平线。**

告别千篇一律的选项与线性结局。NexusTale 将大语言模型（LLM）的无限创造力与经典 TRPG 的严谨规则完美融合，为你打造一个独一无二、永远变化的冒险世界。

## 🎮 为什么选择 NexusTale?

### 🔮 AI 编织的无限世界
每一次对话、每一个决定，都在实时重塑这个世界。
*   **不仅仅是聊天**：AI 扮演的不仅是你的同伴，更是整个世界的“地下城主”（DM）。
*   **拒绝遗忘**：内置智能记忆系统，无论冒险多长，它都记得你们初遇时的那个雨夜。
*   **身临其境**：无论是荒野的呼啸还是废墟的阴森，AI 都会根据你的行动，为你即时描绘最精准的环境氛围。

### 🎲 不再是“过家家”的硬核冒险
我们拒绝单纯的“你画我猜”。这是一场真正的生存游戏。
*   **D20 命运轮盘**：每一个关键行动（探索、战斗、潜行）都需要通过严酷的 D20 判定。运气与策略缺一不可。
*   **Tension System (紧张度系统)**：从“风平浪静”到“生死一线”。当紧张度飙升至 Level 4，每一次错误的决策都可能导致角色死亡。
*   **高风险，高回报**：你敢在满是怪物的废墟中多贪一次搜刮吗？还是带着现有的战利品落荒而逃？

### 🥾 丈量世界的每一步
世界很大，但它真的很远。
*   **拒绝瞬间移动**：在 NexusTale 中，移动是需要时间的。看着地图上的进度条一点点推进，感受长途跋涉的艰辛与期待。
*   **可视化地图**：精美的可视化界面，让你清晰看到距离目的地还有多远，以及沿途潜伏的危险。
*   **室内微观探索**：进入建筑物后，你不再是看一段文字，而是实实在在地推动“搜刮进度条”，直到榨干这里的最后一滴资源。

### 🎬 影音级的沉浸体验
*   **实时绘图**：眼见为实。系统会根据当前的剧情，为你实时生成第一人称视角的精美插画。
*   **动态 BGM**：音乐随剧情而动。探索时的神秘、做饭时的温馨、死斗时的激昂——背景音乐会随着紧张度等级无缝切换。

### ❤️ 跨越次元的羁绊
*   **定制你的专属同伴**：从性别、性格到你们的关系（是相爱相杀的冤家，还是生死与共的战友？）。
*   **情感进化**：随着旅途的推进，你们的羁绊会不断加深，解锁更多专属互动。

## ⚡ 极客引擎 (Tech Highlights)

虽然看起来像个游戏，但因为你的极客之魂，我们还是列一下引擎盖下面的东西：
*   **核心驱动**: Google Gemini Pro (超强上下文理解)
*   **前端架构**: React 18 + Vite + Tailwind CSS
*   **规则逻辑**: TypeScript (严谨的本地状态管理，不依赖 AI 幻觉)

## 🚀 立即启程

只需几步，开启你的传奇：

1.  **下载引擎**
    ```bash
    git clone https://github.com/hewenhan/ai-text-adventure-engine.git
    cd ai-text-adventure-engine
    ```

2.  **准备行囊** (安装依赖)
    ```bash
    npm install
    ```

3.  **注入灵魂** (配置 API Key)
    创建一个名为 `.env` 的文件，填入你的 Google Gemini API Key：
    ```env
    VITE_GEMINI_API_KEY=你的密钥
    ```

4.  **踏上旅途**
    ```bash
    npm run dev
    ```
    打开浏览器访问 `http://localhost:5173`，世界等你探索
